📍 OÙ PLACER VOS CLÉS AWS ?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Créez un fichier ".env.local" à la racine du projet
   (même niveau que package.json)

2. Ajoutez-y ce contenu :

   AWS_ACCESS_KEY_ID=VOTRE_ACCESS_KEY_ICI
   AWS_SECRET_ACCESS_KEY=VOTRE_SECRET_KEY_ICI
   AWS_REGION=eu-north-1
   AWS_S3_BUCKET_NAME=winstory-videos
   
   ⚠️ AWS_REGION : Utilisez la région de VOTRE bucket S3 !
   Vérifiez sur https://s3.console.aws.amazon.com/

3. Remplacez VOTRE_ACCESS_KEY_ICI et VOTRE_SECRET_KEY_ICI 
   par vos vraies clés AWS

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔐 COMMENT OBTENIR VOS CLÉS AWS ?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Allez sur : https://console.aws.amazon.com/iam/
2. Users → Add users
3. Nom : winstory-s3-uploader
4. Cochez "Access key - Programmatic access"
5. Permissions : AmazonS3FullAccess
6. Copiez vos clés (Access Key ID et Secret Access Key)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✨ CE QUI A ÉTÉ IMPLÉMENTÉ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ Upload automatique vers S3 lors de la confirmation :
   - Page Recap B2C (création de campagne)
   - Page Recap Individual (création de campagne)
   - Page Recap Agency B2C (création de campagne)
   - Page Completion Recap (completion de campagne)

✅ Les vidéos sont uploadées dans /pending
✅ Les URLs S3 sont enregistrées dans Supabase
✅ Sécurité : clés AWS côté serveur uniquement

⚠️ PHASE DE DÉVELOPPEMENT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

L'upload S3 se fait actuellement lors de la CONFIRMATION pour 
faciliter les tests.

En PRODUCTION, l'upload sera déplacé au moment du PAIEMENT/MINT
pour éviter les coûts de stockage pour les utilisateurs qui 
confirment mais ne paient pas.
